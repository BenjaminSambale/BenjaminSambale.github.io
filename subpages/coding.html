Some silly spare time coding projects

  • As a teenager around 2000, I wrote a feature-rich Minesweeper clone in Borland Delphi. Unfortunately, I lost the source code, but the binary for Windows still works.
  • For my research in representation theory of finite groups, I wrote numerous algorithms for the computer algebra system GAP. These can be found on another github repo.
  • For the sake of learning Python, I wrote a Mastermind clone with a graphical interface using PySimpleGUI. In connection with my lectures on combinatorial group theory, I developed a permutation puzzle based on the word problem.
